# Docs

Tổng hợp tài liệu dự án AI Content Generator.

## Cấu trúc

| Thư mục                     | Mục đích                                                                | Khi nào tạo                  |
| --------------------------- | ----------------------------------------------------------------------- | ---------------------------- |
| **00_Roadmap**              | Lộ trình học, kế hoạch Sprint, checklist, timeline                      | Ngay khi bắt đầu             |
| **01_Software_Engineering** | Kiến thức nền tảng: SDLC, Agile, Scrum, Git, DevOps...                  | Trước khi làm dự án          |
| **02_Product_Discovery**    | Nghiên cứu ý tưởng sản phẩm, khách hàng, thị trường, đối thủ            | Bắt đầu dự án                |
| **03_Business_Analysis**    | Phân tích nghiệp vụ: Persona, User Story, Use Case, BRD                 | Sau khi xác định ý tưởng     |
| **04_Requirement_Analysis** | Phân tích yêu cầu: SRS, Functional, Non-functional, Acceptance Criteria | Trước khi thiết kế           |
| **05_UX_Design**            | Tài liệu về trải nghiệm người dùng: User Flow, IA, Wireframe            | Trước UI                     |
| **06_UI_Design**            | Thiết kế giao diện: Figma, Design System, Components                    | Sau UX                       |
| **07_Database_Design**      | Thiết kế cơ sở dữ liệu: ERD, Data Dictionary, SQL                       | Trước khi viết Backend       |
| **08_API_Design**           | Kiến thức và thiết kế API: HTTP, REST, Swagger, Postman                 | Trước Backend                |
| **09_System_Design**        | Thiết kế kiến trúc hệ thống: HLD, LLD, UML, Sequence Diagram            | Trước khi code               |
| **10_Architecture**         | Clean Architecture, SOLID, Design Pattern, Repository Pattern           | Khi bắt đầu Backend          |
| **11_Backend**              | Kiến thức Backend: Java, Spring Boot, Security, JPA...                  | Trong quá trình code         |
| **12_Frontend**             | HTML, CSS, JavaScript, React, UI Components                             | Trong quá trình code         |
| **13_AI**                   | Prompt Engineering, LLM, AI API, RAG (nếu có), AI Workflow              | Khi tích hợp AI              |
| **14_Testing**              | Unit Test, Integration Test, API Test, Test Case, Bug Report            | Sau khi hoàn thành chức năng |
| **15_DevOps**               | Docker, CI/CD, Linux, Nginx, Cloud                                      | Trước khi triển khai         |
| **References**              | Link tài liệu, sách, video, website, ghi chú tham khảo                  | Xuyên suốt quá trình         |
